# Quarter Tracker

A simple, visual dashboard that shows what quarter of the year you're in and how far into it you are.

## Features

- Big bold Q1/Q2/Q3/Q4 hero label, color-coded by quarter
- Progress bar showing exact % through the current quarter
- Stats grid: days, weeks, and months elapsed vs. remaining
- Full-year timeline bar with quarter markers
- Dark mode, fully responsive (mobile → desktop)
- Auto-refreshes at midnight — no stale data

## Local Development

No build step required. It's a single `index.html` file with no dependencies.

```bash
# Just open it in a browser
open index.html

# Or serve it locally
npx serve .
```

## Deployment

Hosted on Vercel. To redeploy:

```bash
vercel --prod
```
